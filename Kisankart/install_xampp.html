XAMPP Installation Guide for Kisan Kart

Problem Diagnosis

The current error "Database connection failed" is occurring because the PHP MySQL extensions are not properly installed or configured. This is needed for PHP to connect to the MySQL database.

To fix this issue, we need to install XAMPP, which includes:

  • Apache web server
  • MySQL database server
  • PHP with all necessary extensions
  • phpMyAdmin for database management

Step 1: Download XAMPP

  1. Go to the official XAMPP website: https://www.apachefriends.org/download.html
  2. Download the latest version of XAMPP for Windows
  3. Choose the version with PHP 8.0 or higher

Step 2: Install XAMPP

  1. Run the downloaded installer
  2. You may get a UAC (User Account Control) warning - click "Yes" to proceed
  3. In the setup wizard, click "Next"
  4. Select the components to install. Make sure the following are checked:
    • Apache
    • MySQL
    • PHP
    • phpMyAdmin
  5. Choose the installation folder (default is C:\xampp) and click "Next"
  6. Uncheck "Learn more about Bitnami" if you don't need it
  7. Click "Next" and then "Install" to begin the installation
  8. Wait for the installation to complete
  9. When installation is finished, check "Start Control Panel" and click "Finish"

Step 3: Start Apache and MySQL

  1. In the XAMPP Control Panel, click the "Start" buttons next to Apache and MySQL
  2. Both services should start and their status indicators should turn green
  3. If you get port conflicts, you may need to change the ports or stop conflicting services

Step 4: Verify Installation

  1. Open your web browser and go to http://localhost
  2. You should see the XAMPP welcome page
  3. Click on "phpMyAdmin" in the top menu to verify database access
  4. Create a new database named "kisan_kart" if it doesn't exist

Step 5: Move Your Project Files

  1. Copy your Kisan Kart project files to the htdocs folder (C:\xampp\htdocs\Kisankart)
  2. Make sure all files are in the correct location

Step 6: Test Your Application

  1. Open your web browser and go to http://localhost/Kisankart/
  2. Try to log in to your application
  3. If you still encounter issues, run the MySQL diagnostic tool: http://localhost/Kisankart/mysql_diagnostic.php

Troubleshooting

  • Port conflicts: If Apache or MySQL won't start due to port conflicts, you can change the ports in the XAMPP Control Panel by clicking "Config" next to each service.
  • Missing database: If your database doesn't exist, create it using phpMyAdmin or run your database creation script.
  • Permission issues: Make sure XAMPP has the necessary permissions to access your files and folders.
  • PHP extensions: If specific PHP extensions are missing, you can enable them by editing the php.ini file in the XAMPP installation folder.

Next Steps

After installing XAMPP and verifying that it works correctly:

  1. Go to http://localhost/phpmyadmin to manage your database
  2. Create the necessary tables for your application if they don't exist
  3. Try logging in to your application again
